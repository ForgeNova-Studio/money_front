홈 캐시 프리로드 검증 시나리오

공통 준비
- 로그인 완료 상태
- 최소 1개 이상의 가계부 존재
- 홈 화면 진입 가능

S1. 캐시 있음 + 네트워크 지연
1) 홈 진입 직후 네트워크 지연(프록시/에뮬레이터 네트워크 지연)
2) 앱바 가계부 이름과 월 데이터가 즉시 표시되는지 확인
3) 상단 미니 로딩 인디케이터가 표시되는지 확인
4) 네트워크 응답 후 데이터가 갱신되는지 확인

S2. 캐시 있음 + 네트워크 실패
1) 홈 진입 직후 네트워크 차단
2) 캐시 데이터가 계속 표시되는지 확인
3) 스낵바로 실패 메시지가 노출되는지 확인

S3. 캐시 없음 + 네트워크 정상
1) 캐시 삭제 후 홈 진입
2) 기존 로딩 UI가 표시되는지 확인
3) 응답 후 데이터가 정상 표시되는지 확인

S4. 가계부 전환(캐시 있음)
1) 가계부 A로 진입 후 데이터 확인
2) 가계부 B로 전환(캐시 존재)
3) 전환 직후 캐시 데이터가 즉시 표시되는지 확인
4) 백그라운드 갱신 후 최신 데이터로 변경되는지 확인

S5. 가계부 전환(캐시 없음)
1) 가계부 B 캐시 삭제
2) 가계부 A에서 B로 전환
3) 로딩 UI 표시 여부 확인
4) 응답 후 데이터 정상 표시 확인
