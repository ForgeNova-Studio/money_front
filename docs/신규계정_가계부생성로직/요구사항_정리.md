신규 계정 가계부 자동 생성 요구사항 정리

요청 요약
- 신규 회원가입 및 최초 소셜 로그인 시 DEFAULT 가계부를 자동 생성한다.
- 프론트에서 별도 생성 호출 없이 백엔드에서 처리한다.

생성 규칙
- 이름: 내 가계부
- BOOK_TYPE: DEFAULT
- DESCRIPTION: 기본 가계부
- 멤버 역할: OWNER
- 멤버 카운트: 1
- 나머지 nullable 필드: null

생성 시점
- 회원가입 완료 직후
- 소셜 로그인 최초 유저 생성 직후

백엔드 작업 범위(예정)
- 신규 유저 생성 트랜잭션 안에서 기본 가계부 생성
- 동일 유저에 대한 중복 생성 방지
- 가계부 생성 실패 시 가입/로그인 처리 정책 정리

프론트 영향
- 홈 진입 시 가계부 미존재로 인한 빈 상태 최소화
- 추가 엔드포인트 호출 없음
